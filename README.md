| Command          | Description                                                           |
| ---------------- | --------------------------------------------------------------------- |
| `mvto <file>`    | Opens the TCTEE text editor for a file (like `cd` but for editing).   |
| `rd <file>`      | Reads and prints the contents of a file.                              |
| `stand by!'                                                                              |
| `del <file>`     | Deletes a file from the filesystem.                                   |
| `amiwho`         | Shows the current username (`coolguy11`).                             |
| `restartcomp1`   | Restarts the kernel (resets the OS).                                  |
| `poweroffcomp1`  | Shuts down the OS (halts CPU).                                        |
| `fllist`         | Lists all files currently in the TCOE filesystem.                     |
